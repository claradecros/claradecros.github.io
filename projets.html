Site Web Portfolio

Ce projet a consisté en la création de mon site web personnel responsive pour présenter mes compétences, mon parcours et mes réalisations professionnelles. L'objectif était de fournir une plateforme centralisée et visuellement attrayante.

Technologies Utilisées

  • HTML5
  • CSS3
  • JavaScript (pour la navigation)
  • Font Awesome (pour les icônes)

Fonctionnalités Clés

  • Design entièrement responsive
  • Navigation fluide par onglets dynamiques
  • Formulaire de contact
  • Animations CSS pour l'expérience utilisateur

Défis et Solutions

L'un des principaux défis a été de garantir une compatibilité optimale sur tous les appareils en adaptant la structure CSS. J'ai également développé une logique JavaScript personnalisée pour permettre le changement d'onglets sans recharger la page.
