Site Web Portfolio

Ce projet a consisté en la création de mon site web personnel responsive pour présenter mes compétences, mon parcours et mes réalisations professionnelles.

Technologies Utilisées

  • HTML5 / CSS3
  • JavaScript (navigation dynamique)
  • Font Awesome (icônes)

Fonctionnalités Clés

  • Design responsive (mobile/tablette)
  • Animations CSS fluides
  • Formulaire de contact intégré

Défis et Solutions

L'un des plus gros défis a été de maintenir une structure de code organisée et évolutive. Jongler entre les multiples feuilles de style et les animations a nécessité une rigueur particulière dans l'écriture du CSS pour éviter les conflits visuels.
